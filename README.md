## 🙆‍♀️ 개인 프로젝트
> ### **smileworld**
  * 프로젝트 소개
    - 목적 : 2000년대 가장 인기있는 소셜 미디어였던 싸이월드가 복원되면서 어렸을 적 즐겨 찾던 싸이월드와 같이 아날로그 감성이 묻어난 개인 미니 홈페이지를 구현해보고자 함  
    - 현재 해당 사이트는 모바일 앱 형태로만 운영되고 있어, 최근 복원된 싸이월드 앱, 과거 홈페이지 형태 및 기존에 진행된 프로젝트들을 참고하여 클론코딩 진행
  * 사용기술 
    - ✓ Java
      ✓ SpringBoot 
      ✓ MyBatis
      ✓ React
      ✓ CSS
      ✓ MySQL
